# tasks/T-0001.md — OpenSlide tiler & alignment manifest (≤90 min)
**Goal:** Convert 1 SVS to DZI tiles + `manifest.json` with lattice params and `alignment_ok`.
**AC:** CLI `wsi_tiler.py --input slide.svs --out tiles/slide_id --patch-px 256` writes tiles + manifest with: `slide_id,level0_width,level0_height,mpp0,patch_px,tile_size,overlap=0,anchor=(0,0)`. A checker verifies lattice math correctness by sampling positions at level-0 and confirming round-trips `indexOf(center(i,j)) == (i,j)`, then prints `alignment_ok=true`. 
**Constraints:** Python + `openslide-python`; JPEG Q≈80; `tile_size=512`; `overlap=0`. Grid is zoom-level-fixed: cells = patch_px × patch_px at each zoom.
**Test notes:** Open output in a generic DZI viewer to sanity‑check.
